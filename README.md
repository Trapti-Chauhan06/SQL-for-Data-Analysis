# SQL-for-Data-Analysis
SQL data analysis project using MySQL.

# Objective
The objective of this task is to practice SQL for data analysis by querying an e-commerce dataset using MySQL. The project demonstrates how SQL can be used to retrieve, filter, summarize, and analyze structured data.

# Tools Used
* MySQL
* MySQL Workbench
* GitHub

# Dataset
* Superstore Dataset

# SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* Subqueries
* Views
* Indexes for query optimization

# Files Included

*'task3_queries.sql' – SQL queries used for data analysis.
* 'superstore_dataset.csv' – Dataset used in this project.
* 'screenshots/' – Screenshots showing the execution and output of SQL queries.
* 'README.md' – Project documentation.

# Learning Outcomes
* Retrieved and filtered data using SQL queries.
* Sorted and grouped records for analysis.
* Performed calculations using aggregate functions.
* Combined data from multiple tables using joins.
* Used subqueries for advanced data retrieval.
* Created views to simplify analysis.
* Created indexes to improve query performance.
